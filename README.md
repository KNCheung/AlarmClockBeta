# Alarm Clock #
Just an alarm clock with assistant functions.
Base on [STM32F103R8T6](http://www.st.com/web/catalog/mmc/FM141/SC1169/SS1031/LN1565) and [RT-Thread](http://www.rt-thread.org/) 1.2.1
